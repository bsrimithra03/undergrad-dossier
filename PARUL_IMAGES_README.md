# Parul University Background Images Setup

## Required Images

To complete the setup, please add the following Parul University images to this directory:

### 1. Student Login Background
- **File name:** `parul_university_campus.jpg`
- **Recommended:** Campus view, students, or university entrance
- **Size:** 1920x1080 or higher resolution
- **Format:** JPG or PNG

### 2. Admin Login Background  
- **File name:** `parul_university_admin.jpg`
- **Recommended:** Administrative building, office, or professional university setting
- **Size:** 1920x1080 or higher resolution
- **Format:** JPG or PNG

## How to Add Images

1. Obtain official Parul University images (with proper permissions)
2. Rename them to the exact filenames mentioned above
3. Place them in the same directory as `app.py`
4. Run the application: `py -m streamlit run app.py`

## Fallback

If images are not found, the application will show:
- Parul University branded gradient backgrounds
- Orange (#FF6B35) and gold (#F7931E) color scheme
- Professional styling maintained

## Image Guidelines

- Use high-quality, professional images
- Ensure proper licensing/permissions for university images
- Images should represent Parul University positively
- Consider image file size for web performance
